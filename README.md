# Athens City Guide

A responsive multi-page website developed for the course  
Web Technologies (CN4003).

This project was created as part of a first-year university assignment
and reflects my learning process at that time, with a focus on clean
structure and web fundamentals.

## Overview
The website functions as a simple tourist guide for Athens, providing
visitors with essential information about attractions, restaurants,
accommodation, nightlife, and contact details.

## Technologies
- HTML5
- CSS3
- JavaScript

## Features
- Multi-page navigation
- Responsive design for desktop and mobile
- Image galleries and structured content sections
- Contact form
- Organized file and folder structure

## What I focused on
- Clean and semantic HTML structure
- Proper CSS organization across multiple stylesheets
- Responsive layout using flexible units and media queries
- Performance and fast page loading without external libraries

## Setup
1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. No additional setup or dependencies are required.

This project demonstrates my early approach to web development
and my emphasis on building strong foundations.
